# 知域 ArkTS 组件排版文档

基于 `design-preview/zhiyu-v2-prototype.html` 逐层提取。**本文档只描述组件位置和交互，不指定 ArkTS 实现方式。**

---

## 页面结构总览（从上到下）

```
┌──────────────────────────────┐
│ ① StatusBar                 │  时间 09:41 + 信号/WiFi/电池
├──────────────────────────────┤
│ ② TopNav                    │  沉浸 / 发现  模式切换
├──────────────────────────────┤
│                              │
│ ③ ImmersiveCard (沉浸模式)  │  大卡片，占满剩余空间
│    或                        │
│ ③ DiscoverGrid (发现模式)   │  双列网格
│                              │
├──────────────────────────────┤
│ ④ Pager                     │  页码指示器（仅沉浸模式）
├──────────────────────────────┤
│ ⑤ FloatNav                  │  ‹ 上一张  ·  转发  ·  › 下一张
├──────────────────────────────┤
│ ⑥ Toast（条件显示）          │  操作反馈提示，1.8s 自动消失
├──────────────────────────────┤
│ ⑦ BottomNav                 │  发现 | 收藏 | 附近圈 | 我的
└──────────────────────────────┘
```

## ① StatusBar（状态栏）

**位置：** 页面最顶部  
**高度：** 约 44px（pt:12 pb:4）  
**水平内边距：** 左右各 24px（px:6 = 24px）

**内容：**

| 元素 | 位置 | 样式 |
|---|---|---|
| 时间文字 "09:41" | 左侧 | 12px，fontWeight:600，颜色 `#1F2937` |
| 信号强度图标 | 右侧 | 16×16 SVG，实心矩形柱状图，`#1F2937` fill |
| WiFi 图标 | 右侧 | 16×16 SVG，扇形弧线，`#1F2937` fill |
| 电池图标 | 右侧 | 24×16 SVG，矩形框+内部实心块，`#1F2937` stroke+fill |

**交互：** 无交互，纯展示

**ArkTS 注意：** HarmonyOS 系统自带状态栏，实际 App 通常省略此区域。

---

## ② TopNav（模式切换标签）

**位置：** StatusBar 下方，水平居中  
**上方间距：** pt:16（16px）  
**下方间距：** pb:8（8px）  
**两个标签间距：** gap:24（24px）

**结构：**
```
        ┌──────┐     ┌──────┐
        │ 沉浸  │     │ 发现  │
        │  ·   │     │      │
        └──────┘     └──────┘
```

**标签样式：**

| 属性 | 默认态 | 激活态 |
|---|---|---|
| 字号 | 14px | 14px |
| 字重 | 400（Regular） | 600（SemiBold） |
| 颜色 | `#9CA3AF`（灰） | `#111827`（深灰黑） |
| 激活指示点 | 无 | 4×4px 圆点，`#1F2937`，在文字下方 mt:4 |

**交互：**
- 点击"沉浸" → 切换到 `ImmersiveCard` 单卡片浏览
- 点击"发现" → 切换到 `DiscoverGrid` 双列网格浏览

**ArkTS 实现：** 2 个 Text + 条件渲染的 Circle 指示点，包裹在居中的 Row 中。

## ③ ImmersiveCard（沉浸模式大卡片）

**位置：** TopNav 下方，垂直居中在可用空间内  
**卡片宽度：** 屏幕宽度的 88%（`width: '88%'`）  
**卡片高度：** 卡片区域的 78%（`height: '78%'`）——注意：这个 78% 不是屏幕高度，是 Stack 容器的高度

### 3.1 卡片容器

| 属性 | 值 |
|---|---|
| 圆角 | 28px |
| 内边距 | 左右 24px，上下 20px |
| 边框 | 1px solid `grad.border` |
| 背景 | `linear-gradient(165deg, grad.gradientStart … grad.gradientEnd …)` |
| 阴影（多层，ArkUI 选一层） | `0 14px 32px rgba(160,150,130,0.18)` + 顶部高光线 |

**顶部高光线（card-highlight）：** 卡片顶部 1.5px 高的 Row，距左右边缘各 14px，渐变 `transparent → rgba(255,255,255,0.8) → transparent`

### 3.2 卡片内容从上到下排列

```
┌────────────────────────────────┐
│  # 生物奇观                    │  ← ① 分类标签：11px Medium，accent色，letter-spacing:1.2
│                                │
│  章鱼有三颗心脏，游泳时主心脏  │  ← ② 标题：19px Medium，1F2937
│  会停止跳动。                  │     line-height:1.5，max-lines:3
│                                │
│  ━━━━━                        │  ← ③ 分割线：32px宽×4px高，border色，圆角2px
│                                │
│  这意味着游泳对它们来说是一件  │  ← ④ 正文：13.5px Light，6B7280
│  极其耗费体力的事情。因此，在  │     line-height:1.625，flex-1 撑满
│  深海中，章鱼更喜欢爬行……     │
│                                │
│  ───────────────────────────── │  ← ⑤ 顶部分割线：1px，border色50%透明
│                  ☆   ♥   ↗    │  ← ⑥ 操作按钮：3个，靠右，gap:20
└────────────────────────────────┘
```

### 3.3 操作按钮

**位置：** 卡片底部，水平靠右排列  
**与正文间距：** `border-top` 1px 分割线 + pt:14 上内边距

每个按钮：
| 属性 | 值 |
|---|---|
| 尺寸 | 36×36px |
| 圆角 | 50%（圆形） |
| 背景 | `rgba(255,255,255,0.4)` |
| 边框 | 1px solid `rgba(255,255,255,0.4)` |
| 阴影 | `0 6px 14px rgba(160,150,130,0.14)`（单层） |
| 顶部高光线 | 按钮顶部 1px 白线，距左右各 10px |
| 内部图标 | 22×22px |
| 按下效果 | scale 0.9 |

**三个按钮从左到右：**

| 按钮 | 图标 | 默认颜色 | 激活颜色 |
|---|---|---|---|
| 收藏 | 书签 Path | stroke `#4B5563` | fill+stroke `#F59E0B`（金色） |
| 点赞 | 心形 Path | stroke `#4B5563` | fill+stroke `#EF4444`（红色） |
| 转发 | 3圆+2连线 | stroke `#4B5563` | stroke `#4B5563`（不变色） |

**交互：**

| 操作 | 效果 |
|---|---|
| 收藏点击（未收藏→收藏） | 图标变金色实心 `#F59E0B`，Toast"已添加到收藏" |
| 收藏点击（已收藏→取消） | 图标变空心，Toast"已取消收藏" |
| 点赞点击（未点赞） | 图标变红色实心 `#EF4444`，Toast"已点赞" |
| 点赞点击（已点赞） | Toast"已经点过赞啦"，图标不变 |
| 转发点击 | 复制文本到剪贴板，Toast"已复制到剪贴板，去分享吧" |

### 3.4 卡片后方虚影

在沉浸模式当前卡片的后方，有下一张卡片的虚影，表示"后面还有更多"：

| 属性 | 值 |
|---|---|
| 尺寸 | 与卡片相同（88%×78%） |
| 圆角 | 28px |
| 背景 | 下一张卡片的渐变 |
| 不透明度 | 0.25 |
| 缩放 | 0.95 |
| 向下偏移 | 16px |

**ArkTS 实现：** Stack 中将虚影放在卡片下方，scale 0.95，opacity 0.25

### 3.5 卡片滑动切换

**手势：** 水平方向 PanGesture，最小滑动距离 60px（120px 才触发切换）  
**动画：** 卡片跟随手指移动；手指抬起后，超过阈值则飞出淡出（300ms），不足则回弹（250ms EaseOut）

## ④ Pager（页码指示器）

**位置：** 卡片区域下方，水平居中  
**显示条件：** 仅在沉浸模式 + 卡片数 > 1 时显示  
**容器 padding：** pt:14 pb:10

**样式：**

| 状态 | 形状 | 尺寸 | 颜色 |
|---|---|---|---|
| 当前页 | 横胶囊 | 24×6px，圆角 3px | `#9CA3AF` |
| 非当前页 | 圆形 | 6×6px | `#D1D5DB` |
| 间距 | — | gap:8（dot 间距 16px） | — |

**动画：** 切换时宽度过渡 300ms  
**最大显示：** 10 个点

## ⑤ FloatNav（浮动翻页按钮）

**位置：** Pager 下方，水平居中  
**容器 padding：** pt:2 pb:8

**3 个按钮水平排列：**

| 按钮 | 图标 | 图标规格 |
|---|---|---|
| 上一张 | `< 箭头` | ChevronLeft：`M15 18l-6-6 6-6`，20×20 SVG |
| 转发 | ShareIcon（3圆+2连线） | 20×20 SVG |
| 下一张 | `> 箭头` | ChevronRight：`M9 18l6-6-6-6`，20×20 SVG |

每个按钮：
| 属性 | 值 |
|---|---|
| 尺寸 | 44×44px |
| 圆角 | 50% |
| 背景 | `rgba(255,255,255,0.8)` + `backdropBlur(12)` |
| 边框 | 1px solid `rgba(255,255,255,0.6)` |
| 阴影 | `0 6px 14px rgba(160,150,130,0.14)` |
| 图标颜色 | `#6B7280` |
| 按下效果 | scale 0.9 |
| 按钮间距 | gap:14（左右按钮距中间 gap:14） |

**边界态：**
- 第一张时：左箭头 opacity 0.25 + 禁用
- 最后一张时：右箭头 opacity 0.25 + 禁用

**交互：**
- 左箭头 → 上一张卡片
- 转发 → 同卡片内转发按钮逻辑
- 右箭头 → 下一张卡片

## ⑥ Toast（操作反馈提示）

**位置：** 浮动按钮下方，水平居中（绝对定位，不占布局空间）  
**显示条件：** `this.vm.toastMsg` 非空时显示  
**消失：** 1.8s 后自动清除

| 属性 | 值 |
|---|---|
| 背景 | `rgba(31,41,55,0.88)` |
| 圆角 | 20px（胶囊） |
| 内边距 | 左右 20px，上下 10px |
| 内容 | `✓` 绿色对勾（`#A7F3D0`，16×16）+ 白色文字（14px） |
| 阴影 | 无 |
| backdropBlur | 10 |

**位置：** 页面最底部  
**高度：** 70px  
**上边距：** 无（贴底）  
**结构：** 水平均分 4 列，每列图标+文字垂直排列

| 属性 | 值 |

**位置：** TopNav 下方，占据卡片区域  
**布局：** 2 列网格  
**列间距：** 12px  
**行间距：** 12px  
**容器 padding：** 左右 16px，上 8px，下 96px（为导航留空）  
**可垂直滚动**

| 阴影 | `0 10px 24px rgba(160,150,130,0.15)`（单层） |

## ⑩ 页面整体布局参数（供 Column 层级参考）

```
├── Toast（绝对定位，不占空间，条件渲染）
└── BottomNav（70px）
```
